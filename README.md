# MatchText

MatchText é uma aplicação web desenvolvida em Java com Spring  
O objetivo do projeto é exibir frases aleatórias de séries e filmes, funcionando como um pequeno serviço de curiosidades ou inspiração para o usuário  

## Sobre o projeto
A aplicação tem uma estrutura simples: ao acessar a rota principal, o sistema seleciona uma frase aleatória de uma lista pré-definida e a retorna para o usuário 

## Tecnologias utilizadas
- Java  
- Spring Framework (Spring Web)  

## Funcionalidades
- Retorno de uma frase aleatória a cada requisição  
- Estrutura enxuta para estudo de rotas e controllers  

## Como executar
1. Clone o repositório  
2. Abra o projeto em sua IDE de preferência  
3. Execute a aplicação via Spring Boot  
4. Acesse `http://localhost:8080` para visualizar o resultado
